Projet 1
Initiation au HTML & CSS

Présentation

Ce projet marque une première immersion dans la création de pages web. Il permet d’explorer la structure HTML5, d’utiliser des balises sémantiques et d’appliquer des styles simples grâce au CSS.

Objectifs pédagogiques

  • Comprendre la logique d’un document HTML5.
  • Construire un mini-site multi-pages.
  • Styliser du contenu à l’aide du CSS.
  • Mettre le projet en ligne via GitHub Pages.

Compétences développées

  • Structuration et organisation d’un site web en HTML.
  • Personnalisation visuelle via CSS (mise en page, typographie, couleurs).
  • Responsive design avec media queries.
  • Intégration d’images et de contenus accessibles.
  • Déploiement en ligne et validation W3C.

Rendu final

Le site obtenu comporte une page d’accueil, une galerie de projets ainsi qu’une page détaillée pour chaque réalisation. Il respecte les standards HTML/CSS et s’adapte aux différentes tailles d’écran.
